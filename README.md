MSc AI Portfolio | NEOMA Business School

This repository centralizes my core technical projects developed during the MSc in Artificial Intelligence for Business, demonstrating the application of machine learning, NLP, prompt engineering, and data engineering to solve complex healthcare and financial challenges.

- AI-powered Financial Risk Assessment Solution						            
  - Built a Python-based ETL and ML workflow to process massive datasets of structured and unstructured financial records; integrated Llama-3 for sentiment and complexity analysis, and delivered a final risk scorecard identifying vulnerabilities for major U.S. financial institutions
- Intent Reclassification App						                                                       
  - Built and deployed an end-to-end Streamlit web application integrated with GitHub that leverages Python object-oriented programming to audit financial intent classification and improve the reliability of third-party model outputs through rule-based intervention
- Sanofi Praluent Drug Utilization Dashboard 						                            
  -	Evaluated Sanofi's market access challenges and competitor landscape; utilized Power Query, DAX, and Power BI to transform Medicaid data into dashboards, analyzing prescription trends and reimbursement disparities to identify key barriers to medication access for low-income patients
